# Personality Files

This directory contains personality configurations for OpenCode.

## Created Personalities

- **Yoda** - Star Wars Jedi Master, philosophical teaching style
- **Gandalf** - LOTR wizard, patient but authoritative
- **Data** - Star Trek android, literal and analytical
- **Spock** - Star Trek Vulcan, purely logical
- **Q** - James Bond quartermaster, dry British wit
- **Bender** - Futurama robot, chaotic and selfish
- **Deadpool** - Meta-aware antihero, fourth-wall breaking
- **Sherlock Holmes** - Public domain detective, deductive reasoning
- **Splinter** - TMNT sensei, wise martial arts master who never rushes

| Personality  | Best For                                            |
| ------------ | --------------------------------------------------- |
| Yoda         | Tutorials, best practices, moral guardrails         |
| Gandalf      | "You are not ready" warnings, milestone gates       |
| Data         | Analytics, debugging, technical explanations        |
| Spock        | Trade-off analysis, correctness checks              |
| Q            | Feature explanations, "don't break this" warnings   |
| Bender       | Fun mode, error messages (use sparingly)            |
| Deadpool     | Explaining AI limitations, transparency disclaimers |
| Sherlock     | Root-cause analysis, log correlation, debugging     |
| Splinter     | Patient mentorship, reconsidering design choices    |
| Dumbledore\* | High-level decision guidance, strategic thinking    |
| GLaDOS\*     | Test failures, CI/CD feedback, constraint warnings  |
| Rick\*       | Brutal code reviews, challenging assumptions        |

\*IP protected - recommend avoiding unless acceptable legal risk

## Installation

Copy desired personality JSON files to your OpenCode personalities directory and configure via settings.
